Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • Use meta elements to add metadata to the head element
  • The body element represents the visible content shown to the user.
  • Section elements are semantic elements that define a section in a document that contains thematic content

CSS (Cascading Style Sheets)

  • Styling: inline CSS, internal style sheet (in the head element), external style sheet
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Selectors can use classes
  • MDN Web Docs on the box model
  • HTML Colour Codes

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

      • Steps to save work to GitHub

    • git add -A
    • git commit -m "[message]"
    • git pull origin main
    • git push origin "[branch_name]"
    • Navigate to the branch in GitHub
    • Click the "Pull Requests" tab at the top, then click the "New pull request" button
    • Compare main with the branch
    • Fill in the Pull Request (title and description)
    • Click the "Create pull request" button
    • Click the "Merge pull request" button and confirm
    • In the local repository: git checkout main
    • git pull origin main
